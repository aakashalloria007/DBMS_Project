EcoTrack: An inventory dashboard leveraging Database Management Systems (DBMS), Machine Learning (ML), and Web Development. It provides comprehensive insights into restocking, sales, and inventory management, promoting efficiency and sustainability. EcoTrack streamlines inventory processes, fostering cost savings and environmental responsibility

At the same time, it also helps users to find relevant products and make a marketplace easier for the consumers while making it sustainable, reducing waste, and promoting responsible resource consumption which can be frustrating on other solutions. It helps the environment by avoiding excess inventory leading to reduced costs while also making sure there is no insufficient inventory which can result in loss of sales and customer dissatisfaction. EcoTrack aims to address these problems by providing real-time insights into inventory management and making a marketplace easier for consumers to use while making it sustainable, reducing waste, and promoting responsible resource consumption.
